For at få dette program til at kunne kørt i processing, skal man installere processing herfra "https://processing.org/download" 
Derefter skal man hente zip filen fra dette repository og udpakke den
Derefter går man til "Tools" i den øverste navigationsbar og i dropdown menuen åbner "Manage tools"
Derefter åbner man i det nye vindue som er åbnet i den øverste bar "Libraries"
Derefter søger man under "Filer (forstørrelsesglasset)" efter Sound, og finder den som er lavet af "The Processing Foundation" med et logo til venstre for deres navn
Derefter trykker man på install i bund højre hjørne af vinduet, og venter på den er installeret
Derefter skal man åbne filen "Minecraft_menu.pde" med processing
Derefter kan man åbne selve programmet ved at trykke på start pilen i top venstre hjørne af processing
Så kører programmet
